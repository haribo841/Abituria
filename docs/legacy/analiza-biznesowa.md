# Analiza biznesowa

> Dokument historyczny. Punkty nie oznaczają obecnie zaimplementowanych funkcji.

- Określenie celów projektu z punktu widzenia klienta, potrzeb użytkownika końcowego i modelu biznesowego.
- Zdefiniowanie zakresu, harmonogramu, kamieni milowych i kryteriów akceptacji.
- Wypracowanie modelu licencyjnego.
- Zebranie i zapisanie wymagań zgodnie z wybraną metodyką.
- Opracowanie architektury z uwzględnieniem wybranych technologii.
