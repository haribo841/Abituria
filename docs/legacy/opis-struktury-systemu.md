# Opis struktury systemu

> Dokument historyczny. WPF i SQL Server zostały zastąpione przez AvaloniaUI oraz SQLite.

Pierwotnie zakładano backend C#/.NET i interfejs WPF. Lista pomysłów obejmowała:

- opcjonalne logowanie oparte na bazie danych,
- kalkulator podstawowych działań, nawiasów, potęg i pierwiastków,
- działy matematyki z teorią, zadaniami i stopniowo odsłanianymi podpowiedziami,
- liczby naturalne, indukcję, alfabet grecki, algebrę, równania i nierówności,
- zadania maturalne oraz wprowadzające,
- opcjonalne materiały wideo,
- losowanie zadań z puli maturalnej.

Plan nawigacji obejmował kalkulator, wcześniejsze matury, działy, wideo i osobny segment ćwiczeń. Te pozycje zostały sklasyfikowane w aktywnym planie rozwoju.
